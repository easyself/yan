Git 的一些常用用法笔记

2019-05-10

  1. 本地创建分支 v-1.2

    $ git branch v-1.2
$ git branch
* master
v-1.2

  2. 切换本地分支

    $ git branch
* master
v-1.2

$ git checkout v-1.2
Switched to branch 'v-1.2'

$ git branch
master
* v-1.2

  3. 把本地分支提交到远程分支仓库

    $ git add --all

$ git commit -m "ASO design for Galaxy version 1.2"

$ git push origin v-1.2
